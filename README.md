# Lektionsrepo 3 dec

Nu hoppar vi över till att försöka optimera våra HTML-sidor, och vi inleder med att granska och optimera utifrån ett tillgänglighetsperspektiv. Faktum är att det inte spelar någon roll hur bra våra webbsidor är i ett konverteringsoptimeringsperspektiv, om de inte också är tillgänglighetsanpassade. Ju fler människor som kan använda våra webbsidor, desto fler potentiella kunder har vi. Lägg därtill att det sedan juni 2025 är lagstadgat i Sverige och EU att alla webbsidor behöver uppfylla ett antal krav för att få existera.

## Granska Manuellt

Steg 1 för dagen kommer vara att manuellt granska valfri webbsida. Sitt gärna tillsammans i små grupper så att det blir lite diskussion kring olika delar. Övningen hittar ni mappen [granska-med-wcag](./granska-med-wcag).
