# Granska med verktyg

För denna övning krävs att du laddar ner Wave Evaluation Tool som är en extension till **Chrome**.

## Instruktioner

1. Klona hem hela lektionsrepot, navigera dig till mappen **Qay Insurances** och öppna projektet i en webbläsaren med hjälp av _Live Server_ i VS Code.

2. Låt **Wave** analysera sidan och notera sedan vilka fel som hittas. Gå igenom felen ett i taget och försök förstå vad felet är, samt varför det räknas som ett fel.

3. Åtgärda felen efter bästa förmåga. Använd **Wave** för att läsa på om hur du på bästa sätt läser nuvarande fel.
